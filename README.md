# Описание

## 1. Сервис авторизации (первый коммит)

По строке вида
``localhost:8080/authorize?user=<ИМЯ_ЮЗЕРА>&password=<ПАРОЛЬ_ЮЗЕРА>``
возвращает json-объект, описывающий выданные права пользователя.

В случае, если происходит исключение - выкидывается исключение кастомного класса, 
обработка исключения выполняется с помощью аннотаций в классе Advice.

Тесты запускают приложение в контейнере и к нему делают запросы.
